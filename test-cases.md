# Test Cases

### TC-01 Login with valid credentials

Steps:
1. Open login page
2. Enter valid username
3. Enter password
4. Click Login

Expected result:
User successfully logs into the system

### TC-02 Login with invalid password

Steps:
1. Open login page
2. Enter valid username
3. Enter wrong password
4. Click Login

Expected result:
Error message appears

### TC-03 Add product to cart

Steps:
1. Open product page
2. Click "Add to cart"

Expected result:
Product appears in cart

### TC-04 Remove product from cart

Steps:
1. Open cart
2. Click remove product

Expected result:
Product removed from cart

### TC-05 Checkout process

Steps:
1. Add product to cart
2. Click checkout
3. Fill personal information
4. Confirm order

Expected result:
Order confirmation appears
